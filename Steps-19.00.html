Adding World Cloud Visual 4 (Apps by Job) 19


Steps:

1.     Here we will need a Word Cloud visual if you do not have it you can download it by clicking 'Get more visuals' in Visualizations then search for 'Word Cloud' in the search box.

2.     Click on an empty space on the canvas, and from the visualizations pane select 'Word Cloud' and choose select 'TITLE’ column from the table.

3.     Format, Visual, General, Min repetitions (2), Max words (135), Min font size (20%), Max font size (100%), Word breaking 'On'

4.     Stop words 'On', Rotate text 'On'.
5.     General, Properties, Height (327), Width (358), Position, Horizontal (564), Vertical (393)

6.     Title 'On', Text “Apps by Job”, Font (DIN, 14), Background color(#414FB1), Centre alignment, Text wrap 'On'.

7.     Effects, Background 'On', Color(#EAEBF7), Transparency (70%), Visual border 'On', Color (#B2B8E9).
