Adding Line and Stacked Column Visual 2 (Apps by US Economic Sector) 17

Steps:
1.     Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart

2.     Fill up figures provided below for the necessary fields

Build Visual Tab:
1.     Set the X-axis to "" and " US_Economic_Sector."

2.     Assign the column values to "Apps per Mth."

3.     Assign the line values to "AR", “AR YoY%”, and “Wage YoY%."

4.     Configure the tooltips to display "AR", “AR YoY%”, "Wage YoY%", "Apps per Yr”, and "Apps per Mth YoY%."

5.     Rename the fields as necessary and remove any dot symbols.

6.     Sort the axis by " US_Economic_Sector" in ascending order.

Format Visual Tab: General:
1.     Properties: Set the size to a height of 312 and width of 538.

2.     Position: Horizontal position of 742 and vertical position of 79.

3.     Title: Turn on the title and set the text to "Apps by US Economic Sector" using the font "Segoe UI Bold" with a size of 14. The text color should be #FFFFFF (white), and the background color should be #414FB1 (Navy Blue). Align the title in the center.

4.     Effects: Enable the background effect with a color of #EAEBF7, transparency 70%, and turn on the visual border with a color of #B2B8E9.

Visual:

·         X-axis: Values: Set the font to "Segoe UI" with a size of 9 and a font color of #414FB1 (Navy blue)..

·        Turn off the title.

·         Y-axis: Values: Use the font "Segoe UI Bold" with a size of 9 and a font color of #414FB1 (Navy blue). Display units should be set to "Thousands".

·        Turn off the title.

·         Secondary Y-axis: Range: Enable the align zeros option.

·        Values: Use the font "Segoe UI Bold" with a size of 9 and a font color of #414FB1 (Navy Blue). Display units should be set to "Auto".

·         Turn off the title.

·         Legend: On

·        Gridlines: Turn off horizontal gridlines.

·        Columns: Set the colors to #606BC1 (Persian Blue)

·        Lines: Set the colors for "Wage YoY" to #A38600 (Olive) and "AR YoY%" to #F34A00 (Orange-red), and “AR” to #E8D166 (Peach). Enable the shade area option.

·        Markers: Turn on markers.

·        Shape: Set the colors to #EB895F (Orange).

·         Data labels: Off

·        You can copy the 1st visual and modify the value and position accordingly.
