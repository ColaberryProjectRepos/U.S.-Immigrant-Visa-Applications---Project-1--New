Problems Encountered:

1. DAX Formula: While working on creating measures for KPIs using DAX formulas, I encountered errors and challenges. Fixing these issues took days of considerable time and effort. 2. Power Query Error: While working with Power Query to load the data, I encountered another problem, and it took a long time to fix it. Attempting to delete a column in Power BI caused an issue, with a warning that it might break the query. I also spent about two days dealing with this Power Query Error. After extensive research, I discovered that the problem originated from the source file. One of the rows contained a hash symbol (#####) instead of a number. To address the issue, I replaced the symbol with 0 values, enabling me to proceed with the next step successfully.

3. GIFs-MB: Creating a GIF for step-by-step instruction project was tough too because I had to keep the file size under 5MB/4MB. It took time and effort to figure out how to meet this requirement. However, with practice and wisdom, I overcame these challenges and created the desired GIF within the specified file size. For example, when I created the first visual (Apps per Mth) with step-by-step instructions, I used wisdom. Rather than creating multiple GIFs for a single visual, I prepared and set up the visual in advance, keeping it hidden. Then, during the GIF's video, I revealed the visual and demonstrated the entire process in one go, step by step, staying withing the limited file size of MB successfully.

4. State Map: I also faced challenges while attempting to use the state map feature. However, after conducting thorough research, I discovered the following helpful instructions from Google on how to configure it: I accessed the Power BI File option, then went to Settings > Preview Features. From there, I selected the Shape map Visual Checkbox and restarted Power BI Desktop. By following these steps, I successfully utilized the "Work by State" visual feature.
