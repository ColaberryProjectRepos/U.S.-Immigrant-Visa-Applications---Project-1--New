Steps: Adding Sub Key Performance Indicators (KPIs) visual cards





Steps (App Per Yr):

1.     Go to the "Visualizations" pane and select the "Card" visual. Then, choose the "App Per Yr" Measure for the card.

2.     Under the "Visual" section, adjust the "Callout" value and set the font to "DIN" with a font size of 15. Set the color to "#000000" and set the category label to 'Off'.

3.     Navigate to the "General" section and modify the properties. Set the size to a height of 39 and a width of 74. Adjust the position to have a horizontal value of 0 and a vertical value of 319.

4.     Set the title and background to 'Off'. 

5.     Finally, select the "App Per Yr". Measure at the top of the screen. Ensure that its format is set to Whole number with zero decimal places.

Steps (Applications per Mth YoY %):

1.     Go to the "Visualizations" pane and select the "Card" visual. Then, choose the " Applications per Mth YoY % " Measure for the card.

2.     Under the "Visual" section, adjust the "Callout" value and set the font to "DIN" with a font size of 15. Set the color to "#000000" and set the category label to 'Off'.

3.     Go to the "General" section and modify the properties. Set the size to a height of 39 and a width of 74. Adjust the position to have a horizontal value of 129 and a vertical value of 319.

4.     Set the title and background to 'Off'. 

5.     Finally, select the " Applications per Mth YoY % ". Measure at the top of the screen. Ensure that its format is set to percentage with zero decimal places.
