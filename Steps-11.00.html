Data Modeling 11

Steps: 

1.     Before creating the relationship between the 'Calendar' table and 'US_Perm_Visas' table, make sure that the 'date' column in both tables has the same datatype (Date/Time).

2.     Go to the 'Data Modeling' page in Power BI Desktop 

3.     To create a relationship between the 'Calendar' table and 'US_Perm_Visas' table:

路        Drag and connect the 'date' column of the 'US_Perm_Visas' table to the 'date' column in the 'Calendar' table.

路        Double-click on the line connecting the two columns to open the 'Edit Relationship' window.

4.     In the 'Edit Relationship' window:

路        Ensure that the 'Cardinality' status is set to "Many to One" 

路        Confirm that the 'Cross Filter Direction' status is set to "Single"
